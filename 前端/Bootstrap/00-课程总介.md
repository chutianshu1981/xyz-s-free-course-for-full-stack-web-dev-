### 2.1 Git 基本使用方式

1. 下载：[官方下载地址](https://git-scm.com/downloads)
2. 安装
3. 注册 gitee.com 账号
4. 创建项目
5. 在本地编辑项目，同步到网上
6. 分享项目，供别人拉取
